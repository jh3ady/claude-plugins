# claude-plugins

Marketplace de plugins Claude Code par Jean-Denis VIDOT.

## Installation

Ajouter la marketplace :

```bash
claude plugin marketplace add jh3ady/claude-plugins
```

Puis installer un plugin :

```bash
claude plugin install claude-viral-post@jh3ady-plugins
```

## Plugins disponibles

| Plugin | Description |
|--------|-------------|
| [viral-post](./viral-post/) | Générateur de posts viraux pour les réseaux sociaux |
